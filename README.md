# Bidder
## Auction Website

- Uses mongodb, graphql (apollo) and react.
- Jest for testing
- Testing out use of apollo subscriptions via websocket.

### Server
1. ```cd server```
2. ```yarn install```
3. ```yarn setup```
4. ```yarn develop```
5. Graphiql is available at http://localhost:8060/graphiql

### Client
1. ```cd web```
2. ```yarn install```
3. ```yarn develop```
4. Client is available at http://localhost:8061
